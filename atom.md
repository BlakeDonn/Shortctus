## Editing text

* Join next line to the end of current line
```
⌘ + j
```

* Move current line up or down
```
⌘ + control + up/down
```

* Duplicate current line
```
⌘ + shift + D
```

* Upper/lower case current word
```
⌘ + k, ⌘ + u

⌘ + k, ⌘ + u
```

* Transpose characters (switch location of character on each side of cursor)
```
ctrl + t
```
